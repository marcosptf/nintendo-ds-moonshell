Как поставить автозапуск MoonShell2

Способ автоматического запуска MoonShell2 на Вашем адаптере (не запуская оригинальную прошивку/ОС).
Пожалуйста, перед тем, как продолжить, установите MoonShell2, как указано в инструкции и проверьте, запускается ли программа.

Метод автоматического запуска для DSTT
Чтобы напрямую загружаться в MoonShell2 на DSTT, надо заменить систему DSTT программой MoonShell2
Не используйте этот метод для подделки (клона) DSTT.

Примените патч DLDI, запустив '/misctools/AutoDLDIPatch/AutoDLDIPatch.exe'.
Затем, переименуйте файл 'moonshl2re_TTIO_TTCARD IO Library.nds' в 'TTMENU.DAT', и скопируйте его в корень Вашей SD-флешки.

Метод автоматического запуска для R4

Чтобы напрямую загружаться в MoonShell2 на R4, надо заменить систему R4 программой MoonShell2
Не используйте этот метод для подделки (клона) R4.

Примените патч DLDI, запустив '/misctools/AutoDLDIPatch/AutoDLDIPatch.exe'.
Затем, переименуйте файл 'moonshl2_R4TF_M3Simply-R4DS (MicroSD Card).nds' в 'DEFAULT.NDS', и скопируйте его в корень Вашей SD-флешки.

Метод запуска на R4 при помощи значка "Мультимедиа"

Чтобы запускать MoonShell2 на R4 при помощи значка "Мультимедиа", заменять систему R4 не требуется.
Не используйте этот метод для подделки (клона) R4.
Вы можете удалить старый Moonshell.

Примените патч DLDI, запустив '/misctools/AutoDLDIPatch/AutoDLDIPatch.exe'.
Затем, переименуйте файл 'moonshl2_R4TF_M3Simply-R4DS (MicroSD Card).nds' в '_DS_MSHL.NDS', и скопируйте его в корень Вашей SD-флешки.
