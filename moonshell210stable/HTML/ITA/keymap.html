Pulsanti della lista file.
Su / Giù / Sinistra / Destra Muovi il cursore.
Pulsante A Apri il file selezionato.
Tieni premuto A Su un file di musica apre il Player Audio.
Pulsante B Ferma la musica.
Pulsante X / Y Regola il volume.
Pulsante START Apri il menù di sistema.
Tieni premuto START Elimina il file selezionato.
Pulsante SELECT Vai alla modalità di Avvio Rapido.
Tieni premuto SELECT Cattura un'istantanea dello schermo(Escluso il DSi).

Tieni premuto L o R nella lista file...
Touchscreen Avvia lo Strumento Appunti.

Tieni premuto L nella lista file...
Sinistra Mostra la barra di controllo MP3.
Destra Canzone successiva.
Su Pausa
Giù Cambia la modalità di riproduzione.
Pulsante A / B Canzone successiva / precedente.
Pulsante X / Y Regola il livello di luminosità .

Tieni premuto R nella lista file...
Sinistra / Destra Avanza o retrocedi dell' 1% nella ricerca MP3.
Su / Giù Avanza o retrocedi del 5% nella ricerca MP3.
Pulsante A Avvia tutti i file nella cartella corrente.
Pulsanti L + R + A Avvia tutti i file nell'intera flashcard.

Combinazioni L / R nella lista file.
Premi insieme L + R Pausa
Pressione veloce + lunga L / R Avanza o retrocedi dell' 1% nella ricerca MP3.
Premi 2 volte L / R Canzone precedente / successiva.
Premi 3 volte L Pausa/Avvia canzone.
Premi 3 volte R Attiva modalità stand-by.

Pulsanti esterni nella lista file (solo con connettore GBA).
Pressione lunga Avanza dell' 1% nella ricerca MP3.
Pressione veloce + lunga Retrocedi dell' 1% nella ricerca MP3.
1 click Canzone successiva.
2 click Canzone precedente.
3 click Attiva modalità stand-by.
