•휴지통

메모리의 루트에 'dustbox'라는 폴더를 만드시면 파일의 삭제(휴지통으로 이동)를 하실 수 있게 됩니다。

삭제를 원하시는 파일에 커서를 위치하신 후 START 버튼을 길게누르면 확인창이 뜨고 확인창의 내용에 따라 L+R+A 버튼을 누르시면 삭제가 됩니다。
B 버튼을 누르시면 취소됩니다。
안전을 위해 휴지통으로 파일을 옮길 뿐 실제로 삭제하지는 않습니다。실질적인 삭제는 Windows 에서 해주세요。
Windows의 휴지통비우기로 삭제를 하실 수 없습니다, 내컴퓨터(또는 탐색기)에서 메모리를 선택하신 후
dustbox 폴더를 여신 후 내용을 삭제해 주세요。
휴지통내에 같은 이름을 가진 파일이 존재하면 덮어쓰게 됩니다。

「폴더의 이동」,「휴지통폴더 내에서 삭제」,「시스템폴더(moonshl2) 내에서 삭제」등은 에러메세지를 출력한 후 취소됩니다。

휴지통 폴더를 문쉘의 목록창에 안보이게 하려면 /moonshl2/hidepath.ini 파일의
;/dustbox
항목을
/dustbox
와 같이 ” ;” 를 삭제하시면 됩니다。

휴지통 폴더안의 파일도 일반 파일과 마찬가지로 재생이 가능합니다。

파일삭제를 자주할 경우 L+R+A 키가 번거롭다 생각되면 파일목록 환경설정에서 삭제 버튼을 X+A로 변경을 체크해 주세요。
