Использование блокнота

Если вы создадите в корне флэшки папку '/moonmemo', функция блокнота будет активизирована.
Когда функция блокнота активизирована, блокнот можно открыть двумя способами.

1. Зажав кнопку R или L, кликните по тач-скрину.
2. Нажав кнопку START, выберите пункт меню [Открыть блокнот].

В первом случае, начнётся создание первой заметки.
После редактирования заметки, когда Вы закроете блокнот, будет создан файл BMP с названием в виде даты и времени. Невозможно создавать записи с одинаковым названием.

Во втором случае, откроется список существующих записей.
Выберите нужную запись и нажмите А, чтобы её отредактировать.
Если записей не найдено, блокнот откроется в режиме редактирования новой заметки.
Если нажать на кнопку START в списке сохранённых заметок, выбранный файл будет отправлен в корзину.

Когда редактируется какой-то файл BMP, сохранение делаетсея в формате 24 бит, без сжатия, не принимая в расчет первоначальный формат файла.

Возможно открыть любой файл bmp, если он находится в папке '/moonmemo', но так как эта программа не предназначается для отображения изображений, помните, что это - блокнот.
Файлы BMP, используемые в качестве записной книжки, должны быть следующего формата: Windows, 1/4/8/24bit, максимум 256x192 пикселей и без сжатия. BMP в формате ОС/2 не распознаются.
