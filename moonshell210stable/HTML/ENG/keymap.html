Key map of file list.
Up / Down / Left / Right Move cursor.
A button Open file.
A button long. When music file, Open music player.
B button Stop music.
X / Y button Change volume.
START button Open system menu.
START button long. Delete this file.
SELECT button Open sub menu.
SELECT button long. Get screen capture (Excluding DSi)

File list. L or R pressing...
Touch screen The note pad is begun in the new making mode.

File list. L pressing...
Left Show MP3 control panel.
Right Next music.
Up Toggle pause
Down Change play mode.
A / B button Previous or next music.
X / Y button Change backlight level.

File list. R pressing...
Left / Right Previous or next 1% seeking.
Up / Down Previous or next 5% seeking.
R + A button Play all files in folder.
L + R + A button Play all files in disk.

File list. L / R button combinations.
L + R push same time Toggle pause
L / R single + long Previous or next 1% seeking.
L / R double click Previous or next music.
L triple click Toggle pause
R triple click Start stand-by mode.

File list. External button.
Long Next 1% seeking.
Single + long Previous 1% seeking.
Single click Next music.
Double click Previous music.
Triple click Start stand-by mode.
