Утилиты.

LanguageFileChecker.exe Проверяет файлы переводов.
Папка Additional Файлы, подключающие дополнительные функции.
Папка AutoDLDIPatch Программа, создающая файл moonshell2.nds для адаптеров, не поддерживающих Auto DLDI patch.
Папка dpgtools Видео конвертер.
Папка ImageViewer Создатель альбомов/комиксов (IPK).
Папка PluginsText Текстовые файлы, относящиеся к использованным плагинам.
Папка SkinTool Программа для создания скинов.
Папка sndeff Конвертер звуковых эффектов.
Папка SoundFont8bit Набор файлов низкого качества и маленького размера для софтового MIDI эмулятора.
