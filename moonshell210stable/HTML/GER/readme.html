Bitte führe die setup.exe aus und starte anschließend am DS die moonshl2.nds- Datei.

Unterstützte Dateiformate.

Musikformate: MP1, MP2, MP3, OGG, WAV, M4A, AAC, WMA, TTA, MID, RCP, R36, FLAC
MOD-Formate: MOD, IT, MTM, S3M, XM, 669, MED, STM, AMF, GDM, ULT, UNI, ASY, IMF, OKT, STX
Chiptune-Formate: SPC, NSF, GBS, HES, AY, SAP, KSS
Playlist-Formate: M3U, WPL
Bild-Formate: JPG, BMP, GIF, PNG, PSD
Unterstützung für DPG-Filme.
Unterstützung für Text-Dateien.

Einschränkungen bezüglich Formate.

NDS: Keine kommerziellen NDSROMs.
WMA: Keine Unterstützung für 'loss-less'- und 'voice mode'- WMA Dateien.
M4A, AAC, WMA: nur unverschlüsselt.
MID: Keine Unterstützung für das 'Standard MIDI file format.2'- Format.
SPC: Schlechte Tonqualität.
Chiptune: Nur der erste Titel wird abgespielt. Der FM Soundchip kann nicht emuliert werden.
Playlist: Unterstützung der Codierungen ANSI, S-JIS, UTF-8, CP437, CP850, CP1252.
Textdateien: Unterstützung der Codierungen ANSI, S-JIS, UTF-8, UTF-16BE, UTF-16LE. Nur automatische Erkennung.

Nur bei MP3, Jpeg, BMP, PSD und DPG werden Dateiinformationen im Dateibrowser angezeigt.

Größeneinschränkungen.

Bildformate: Auflösungen größer als 3648x2736 Pixel können nicht dargestellt werden.
Textdateien: Dateien mit mehr als 131072 Zeilen werden abgelehnt.
MOD: Dateien über 500KByte können möglicherweise nicht abgespielt werden.
MID, RCP, R36: Dateien über 500KByte können möglicherweise nicht abgespielt werden.
