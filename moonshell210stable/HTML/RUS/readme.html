Установка/Деинсталляция.
Под Windows, настоятельно рекомендуется форматировать вашу карту памяти при помощи программы SDFormatter.
Для наилучшей функциональности, убедитесь, что на карте памяти осталось по крайней мере 60Мб свободного места.
Пожалуйста, скопируйте файл 'moonshl2.nds' и папку '/moonshl2' в корень флешки и запустите 'moonshl2.nds'.

ВНИМАНИЕ! для адаптеров DSTT, R4 и M3 Simply, в случае если НЕ работает обычный способ запуска,
ВМЕСТО файла 'moonshl2.nds' следует использовать файлы 'moonshl2re_TTIO_TTCARD IO Library.nds' и 'moonshl2_R4TF_M3Simply-R4DS (MicroSD Card).nds' соответственно (для удобства их можно переименовать в 'moonshl2.nds').
Получить эти файлы можно применив патч DLDI с помощью программы '/misctools/AutoDLDIPatch/AutoDLDIPatch.exe' (После успешного завершения работы программы, файлы появятся в той же папке).

Для того, чтобы деинсталлировать программу, удалите файл 'moonshl2.nds' и папку '/moonshl2' с карты памяти.

В случае возникновения проблем, обращайтесь к ЧаВо или задавайте вопросы на Форуме GBX.RU

Поддерживаемые форматы.

Аудио: MP1, MP2, MP3, OGG, WAV, M4A, AAC, WMA, TTA, MID, RCP, R36, FLAC
MOD: MOD, IT, MTM, S3M, XM, 669, MED, STM, AMF, GDM, ULT, UNI, ASY, IMF, OKT, STX
Chiptune: SPC, NSF, GBS, HES, AY, SAP, KSS
Плейлисты: M3U, WPL
Изображения: JPG, BMP, GIF, PNG, PSD
Видео: DPG
Тексты: TXT

Ограничения, относительно форматов.

NDS: Не поддерживаются коммерческие ромы.
WMA: Не поддерживаются форматы 'loss-less' и 'voice mode' WMA файлов.
M4A, AAC, WMA: Не поддерживаются зашифрованные файлы.
MID: Не поддерживается формат 'Standard MIDI file format.2'.
SPC: Качество тона очень плохое.
Chiptune: Воспроизводится только первый трек (или первая дорожка). Чип "FM sound" сэмулировать невозможно.
Playlist: Поддерживаются кодировки ANSI, S-JIS, UTF-8, CP437, CP850, CP1252.
Text file: Поддерживаются кодировки ANSI, S-JIS, UTF-8, UTF-16BE, UTF-16LE.

Инфо о файле в проводнике (списке файлов) отображается только о файлах MP3, Jpeg, BMP, PSD, DPG.

Ограничения, относительно файлов большого размера.

Изображения: Размер, превышающий 3648x2736 пикселей (10 Мп) отобразить невозможно (За исключением GIF).
Текстовые файлы: Строки, выходящие за пределы 131072 линий, игнорируются.
MOD: Файл, превышающий размер около 500KБ, может не воспроизвестись.
MID, RCP, R36: Файл, превышающий размер около 500KБ, может не воспроизвестись.

Ограничения для объёмных файлов.

Сокращено воспроизведение файлов формата WMA более, чем 128 кбит/с, для использования программных средств MIDI и для воспроизведения SPC, потому что это требует очень много ресурсов.
Jpeg требуется много времени для отображения (30 секунд для 3х мегапиксельного изображения).
Могут возникать многочисленные рывки при одновременном воспроизведении MIDI и просмотре изображения.
