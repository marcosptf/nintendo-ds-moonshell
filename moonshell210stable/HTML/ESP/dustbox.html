Uso de la Papelera

Cuando se crea una carpeta llamada /dustbox en la raíz de la tarjeta, es posible borrar archivos (mover a la papelera).

La Papelera no se crea automáticamente para evitar problemas.
No cree una papelera en las demás carpetas (incluso en /moonshl2), porque sólo se reconocerá si se encuentra en la raíz.
El nombre de la carpeta /dustbox puede ser una mezcla de mayúsculas y minúsculas.

Para eliminar un archivo, coloque el cursor sobre él y pulse START durante algún tiempo. Aparecerá un mensaje de confirmación. Entonces, pulse los botones L+R+A para borrar el archivo. En caso contrario, pulse el botón B para cancelar.
Para evitar problemas, los archivos no son realmente borrados, sólo se trasladan a la carpeta /dustbox. El contenido de la papelera se puede eliminar definitivamente sólo con el ordenador.
Para vaciar la papelera de Moonshell no utilice la opción "Vaciar papelera" de su ordenador. Seleccione los archivos a eliminar y suprímalos.
Al eliminar un archivo que tiene el mismo nombre que otro que se encuentre en /dustbox, el antiguo será sobreescrito.

Si intenta borrar una carpeta, un archivo en la carpeta Papelera o un archivo en /moonshl2, aparecerá un mensaje de error y, a continuación, la operación se cancelará.

Los archivos en la papelera pueden ser examinados.

Es posible ocultar la carpeta Papelera (para que MoonShell2 no pueda manipularla) cambiando el archivo de configuración /moonshl2/hidepath.ini

Para hacerlo, reemplazar:
;/dustbox
por
/dustbox

Si elimina archivos a menudo, y encuentra molesto tener que pulsar L+R+A cada vez, por favor, consulte /misctools/Additional/EasyDele.txt.
