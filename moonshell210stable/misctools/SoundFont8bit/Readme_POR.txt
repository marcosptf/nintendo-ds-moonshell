
sc88proe.bin é uma fonte de software MIDI de baixa qualidade (8bit) que possui pouco espaço.

Ao copiar este arquivo para a pasta /moonshl2/mididata/sc88proe.bin você poderá poupar cerca de 7MByte.
Você poderá encontrar problemas no som ao reproduzir músicas complexas.

Se você não estiver usando arquivos MIDI, você pode apagar a pasta /mididata.
