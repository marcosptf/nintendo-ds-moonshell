•메모장

moonmemo 폴더를 (moonshl2 폴더속이 아님） 메모리의 루트에 생성하시면 메모장 기능이 활성화 됩니다。
파일목록창에서 L 또는 R 버튼을 누르면서 아래화면을 터치, 또는 START 버튼을 누르신 후 시스템메뉴에서 「메모장을 엽니다」를 선택하시면 됩니다。

파일목록에서 직접 시작할 경우 새파일 작성으로 시작합니다。
메모장을 닫을 때 날자와 시간이 포함된 BMP 파일을 자동으로 저장합니다。 파일명은 지정하실 수 없습니다。

시스템메뉴에서 「메모장을 엽니다」를 선택하신 경우 저장된 메모장의 목록을 보여줍니다。
저장된 메모장 파일이 없는 경우 새파일 작성이 시작됩니다。
목록중에서 A 버튼을 누르시면 선택하신 파일을 편집모드로 엽니다。
목록표시중 START 버튼을 누르면 새파일 작성이 시작됩니다。
목록표시중 START 버튼을 길게 누르시면 파일을 삭제하실 수 있습니다。（휴지통에 옮기는 것이 아닌 직접 삭제합니다）

편집이 완료된 BMP 파일은 원래의 포맷에 관계없이 24bit 무압축 BMP 파일로 자동 저장됩니다。

moonmemo 폴더안에 있는 일반 BMP 파일도 열 수 있습니다만 이미지보기 프로그램이 아닌 메모장 기능에 맞춰 사용해 주세요。
메모장으로 불러올 수 있는 BMP 포맷은 Windows용 256x192 픽셀 이하 1/4/8/24bit 무압축 BMP 파일들 입니다。
OS/2 용의 BMP 파일은 여실 수 없습니다。
파일목록에서 A버튼을 길게 누름으로서 메모장 BMP파일을 읽어오면 편집모드로 작동되고 항상 덮어쓰기 되므로 보존할 파일을
망치지 않게 주의하셔야 합니다。
