Использование корзины

Когда Вы создаете в корне флэшки папку '/dustbox', становится становится возможной функция удалять файлы (перемещать в корзину).

Корзина не создаётся автоматически, чтобы избежать каких-либо проблем.
Не создавайте корзину в других папках (даже в '/moonshl2'), так как она распознаётся только если она находится в корне карты памяти.
Название папки '/dustbox' может быть написано любыми буквами (большими/маленькими/сМЕшаНнЫмИ), но только латинским (лучше - английским) шрифтом.
Чтобы папка не мешала комфортному использованию Moonshell2, можно сделать её скрытой (с помощью компьютера).

Чтобы удалить файл, поместите курсор на него, затем зажмите кнопку START на некоторое время. Появится сообщение о подтверждении. Как только оно появится, нажмите на кнопки L+R+A, чтобы удалить этот файл, или нажмите на кнопку B, чтобы отменить удаление.
Во избежание каких-либо проблем, файлы не удаляются с флэшки, а перемещаются в папку '/dustbox'. Содержание корзины может быть стерто только с помощью компьютера.
Чтобы очистить корзину Moonshell2, не используйте функцию Windows "Очистить корзину". Выделите файлы и удалите их в-ручную.
Когда удаляется файл, у которого название такое же, как у другого файла, уже находящегося в папке '/dustbox', новый файл копируется поверх старого (старый файл затирается).

Если вы попытатесь удалить папку корзины или папку (или файл из папки) '/moonshl2', появляется сообщение об ошибке и операция отменяется.

Если корзина не скрыта, файлы, находящиеся в ней, можно открыть.

Возможно скрыть папку корзины (чтобы MoonShell2 не отображал эту папку и её содержимое), изменив параметры файла '/moonshl2/hidepath.ini' (Метод, альтернативный скрытию папки средствами Windows).

Для этого, замените строку в файле '/moonshl2/hidepath.ini':
';/dustbox'
на
'/dustbox'

Если вы часто удаляете файлы и Вы находите неудобной необходимость нажимать L+R+A каждый раз, пожалуйста, обратитесь к файлу '/misctools/Additional/EasyDele.txt'.
