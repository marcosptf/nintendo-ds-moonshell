
sc88proe.bin, versión baja calidad/ahorro de espacio (8 bits) de instrumentos de sonido MIDI.

Copie a / Moonshl2/mididata/sc88proe.bin para ahorrar alrededor de 7MByte.
La versión 8bit consume pocos recursos, y puede resolver los problemas con el sonido que se desactiva durante la reproducción de canciones complejas.

Si no está usando instrumentos MIDI, puede eliminar la carpeta mididata.
