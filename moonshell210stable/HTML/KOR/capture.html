•화면저장 방법

원하시는 화면에서 SELECT 버튼을 3초 이상 계속 누르시면 화면을 저장하실 수 있습니다。
저장이 끝난 후에는 닌텐도가 멈추게 됩니다만 리더기로 메모리를 연결하신 후 Windows 에서
 ??_pmain.bmp, ??_pover.bmp, ??_psub.bmp 등의 파일을 확인 하실 수 있습니다。
여러 개의 파일을 저장하시게 되면 파일의 이름에 숫자를 붙여가며 저장하게 됩니다。
너무 큰 이미지를 연 상태에서 저장을 하게 되면 메모리 부족으로 인해 저장이 안될 수도 있습니다。

DSi에서 저장을 원하실 경우 전체 환경설정에서 「DSi의 화면저장 기능을 사용」을 체크해 주세요。
