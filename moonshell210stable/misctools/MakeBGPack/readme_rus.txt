
Данная программа создаёт набор "обоев, меняющихся случайным образом".
Cкопируйте созданный файл "bgpack.dat" в папку "moonshl2" на флешку.

Для прекращения автосмены обоев, удалите с флешки файл "/moonshl2/bgpack.dat".
