・インストール / アンインストール / アップデート の仕方

SDFormatterでフォーマットしたSDカードを強く推奨します。
安全のため、多少の空き容量（64MByte以上推奨）を残しておいてください。

インストールは、setup.exeを実行してください。従来の手動コピーでもインストールできます。

アンインストールは、moonshl2フォルダと、moonshl2.ndsファイルを、SDカードから削除してください。

アップデートは、setup.exeを実行してください。古いmoonshl2をバックアップフォルダに移動してからインストールします。

・起動しないときは

SDFormatterでフォーマット後にインストールして動作を確認してください。
他の自作アプリも起動できないときは、ファームウェアなどを確認して下さい。
MoonShell2だけ起動できないときは、よくある質問 を参照してください。

・対応フォーマットリスト

音楽関係の MP1, MP2, MP3, OGG, WAV, M4A, WMA, TTA, MID, RCP, R36, FLAC に対応。
MOD関係の MOD, IT, FAR, MTM, S3M, XM, 669, MED, STM, AMF, GDM, ULT, UNI, ASY, IMF, OKT, STX に対応。
ChipTune関係の SPC, NSF, GBS, HES, AY, SAP, KSS に対応。
プレイリスト関係の M3U, WPL に対応。
画像関係の JPG, BMP, GIF, PNG, PSD に対応。
動画再生(DPG)に対応。
テキストファイル表示に対応。

・フォーマットに関する制限事項

NDS: 商用NDSROMは起動できません。
WMA: ロスレスモードとボイスモードは再生できません。
M4A, WMA: 暗号化ファイルは再生できません。
MID: Standard MIDI file format.2 は再生できません。
SPC: SPC-700エミュレーションは非常に音質が悪いです。
ChipTune関係: FM音源チップは未対応です。
FLAC: 32bits/sampleフォーマットは未対応です。
プレイリスト関係: ANSI, S-JIS, UTF-8, CP437, CP850, CP1252 以外は扱えません。
プレイリスト関係: 複数トラックを含むChipTuneサウンドファイルのプレイリストは"KbMedia Player for Windows Version 2.41.2005.0711"以降で出力されたM3Uファイル以外扱えないと思います。
テキストファイル: ANSI, S-JIS, UTF-8, UTF-16BE, UTF-16LE 以外は扱えません。

ファイルリストに情報が表示されるのは、MP3, Jpeg, BMP, PSD, DPG のみです。

・サイズが大きいファイルに関する制限事項

画像関係: 約3648x2736pixels（約1000万画素）を超えるファイルは開けないことがあります。
テキストファイル: 容量制限はありませんが、131072行を超えた行は無視されます。
MOD関係: 約500KByteを超えるファイルは再生できないことがあります。
MID, RCP, R36: 約500KByteを超えるファイルは再生できないことがあります。

フォルダ名とファイル名の合計（たとえば"/test/mp3files/samplemusic.mp3"だと30文字）が255文字を超えるファイルは扱えません。

・重いファイルに関する制限事項

非常に重いので、128kbps以上のWMAファイルを再生中とソフトウェアMIDI音源を使用中は、画面表示を簡略化します。
プログレッシブJpegの展開には長い時間が掛かります。Bボタンによる中断はできません。（約300万画素で約30秒）
MIDI再生と画像表示を同時に行うと、動的なメモリの奪い合いが発生するため（一応再生できますが）音飛びが頻発します。
