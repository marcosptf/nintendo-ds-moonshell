•DSTT다이렉트 실행 방법

DSTT에서 MoonShell2로 직접 부팅하는 방법입니다。짝퉁 DSTT 에는 시도하지 말아주세요。
이 방법은 DSTT의 커널을 바꾸기 때문에 다른 메뉴는 사용할 수 없게 됩니다。

'/misctools/AutoDLDIPatch/AutoDLDIPatch.exe' 를 실행후 OK버튼을 누르신 후 Create 버튼을 선택해 주세요
생성된  'moonshl2_TTIOSDHC_TTCARD IO Library(SDHC).nds' 를 'TTMENU.DAT' 로 이름을 바꾼 후 메모리의 루트 폴더에 덮어쓰기 해주세요。

•R4다이렉트 실행 방법

R4에서 MoonShell2로 직접 부팅하는 방법입니다。짝퉁  R4에는 시도하지 말아주세요。
이 방법은 R4의 커널을 바꾸기 때문에 다른 메뉴는 사용할 수 없게 됩니다。

'/misctools/AutoDLDIPatch/AutoDLDIPatch.exe'를 실행후 OK버튼을 누르신 후 Create 버튼을 선택해 주세요。
생성된  'moonshl2_R4TF_M3Simply-R4DS (MicroSD Card).nds' 를 'DEFAULT.NDS' 로 이름을 바꾼 후 메모리의 루트에 복사하세요。

•R4 멀티미디어 버튼 사용 방법

R4에서 중간의 멀티미디어 버튼으로 MoonShell2 를 실행하는 방법입니다。짝퉁 R4 에서는 시도하지 말아주세요。
이 방법은 커널과 상관이 없습니다。
이전 문쉘（moonshl폴더）은 필요가 없으므로 삭제하셔도 됩니다。

DLDI 패치는 하지 않으셔도 됩니다. 'moonshl2.nds'를 '_DS_MSHL.NDS' 로 이름을 바꾸고 메모리의 루트에 복사하세요。

•EDGE 멀티미디어 버튼 실행 방법

1) MoonShell2.0의 압축을 해제한 후 moonshl2폴더를 메모리의 루트에 복사합니다。
2) 메모리의 루트에 있는 moonshl폴더속의 모든 파일을 삭제(moonshl폴더는 삭제하시 마세요）합니다。
3) moonshl2.nds를 moonshell.nds로 이름을 바꾸신 후 메모리 루트에 있는 moonshl폴더에 복사합니다。
※ EDGE의 멀티미디어 버튼은 오직 /moonshl/moonshell.nds를 실행해 주는 기능만을 하기 때문에
    어떤 NDS파일이라도 moonshell.nds로 이름을 바꾸시면 실행을 시킬 수 있게 됩니다。 다만 파일이 지정한
    PATH는 변경이 불가능 하기 때문에 moonshl폴더는 반드시 있어야만 합니다。

•CYCRO DS(FW 1.55 Beta2 이전)로 MoonShell 1.71+1과 같이 사용할 때 MoonShell2.0을 멀티미디어 버튼으로 실행
(leopon님께서 알려주셨습니다. 감사드립니다。)

1) MoonShell2.0의 압축을 해제한 후 moonshl2폴더를 메모리의 루트에 복사합니다。
2) 메모리의 루트에 있는 moonshl폴더속의 moonshel.nds를 moonshell171.nds등의 이름으로 변경합니다。
3) moonshl2.nds를 moonshell.nds로 이름을 변경한 후 메모리 루트에 있는 moonshl폴더에 복사합니다。
※ MoonShell1.71+1를 실행하실 때는 CYCRO DS의 파일목록에서 이름을 변경하신 파일을 실행하세요。
※ moonshell171.nds등으로 변경되어진 파일은 어느 폴더에 있더라도 실행이 가능합니다。

•CYCRO DS(FW 1.55 Beta2 이전) 멀티미디어 버튼 사용 방법(MoonShell1.71+1은 삭제했을 때)
(leopon님께서 알려주셨습니다. 감사드립니다。)

1) MoonShell2.0의 압축을 해제한 후 moonshl2폴더를 메모리의 루트에 복사합니다。
2) 메모리의 루트에 있는 moonshl폴더속의 모든 파일을 삭제(moonshl폴더는 삭제하시 마세요）합니다。
3) moonshl2.nds를 moonshell.nds로 이름을 바꾸신 후 메모리 루트에 있는 moonshl폴더에 복사합니다。
※ CYCRO DS의 멀티미디어 버튼은 오직 /moonshl/moonshell.nds를 실행해 주는 기능만을 하기 때문에
어떤 NDS파일이라도 moonshell.nds로 이름을 바꾸시면 실행을 시킬 수 있게 됩니다。 다만 파일이 지정한
PATH는 변경이 불가능 하기 때문에 moonshl폴더는 반드시 있어야만 합니다。

•CYCRO DS(FW 1.55 Beta3 이후) 멀티미디어 버튼 사용 방법
(leopon님께서 알려주셨습니다. 감사드립니다。)

1) setup.exe를 사용해 MoonShell2.0을 메모리에 설치합니다。
2) 메모리의 루트에 있는 CycloDS 폴더속의 config.xml을 문서편집기(메모장등)로 엽니다。
3) config.xml 위에서부터 21번째 줄의 <moonshellPath>/moonshl/moonshell.nds</moonshellPath>를
    <moonshellPath>/moonshl2.nds</moonshellPath>로 수정합니다。
4) CycloDS를 켠 후 미디어버튼을 누르면 문쉘 2.0이 실행됩니다。
