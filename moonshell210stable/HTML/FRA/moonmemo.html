Utilisation du Bloc-notes

Lorsque vous créez le dossier /moonmemo la racine de votre carte flash, le Bloc-notes est activée.
Dès lors, dans le navigateur de fichiers, il est possible d'ouvrir le Bloc-notes de deux façons.

1/ Tout en pressant R ou L, touchez l'écran tactile.
2/ En pressant le bouton START puis en sélectionnant [Ouvrir le Bloc-notes] dans le menu.

Dans le premier cas, l'édition d'un nouveau mémo commence de suite. Vous pouvez écrire votre pense bête.
Après édition, lorsque vous fermerez le Bloc-notes, un fichier BMP sera créé avec comme nom : la date et l'heure. Il n'est pas possible de mettre un nom soi-même.

Dans le second cas, la liste des mémos créés s'affichera.
Sélectionnez un mémo puis appuyez sur A pour l'éditer.
Si aucun fichier n'est trouvé, l'édition d'un nouveau mémo commencera directement.
Si l'on appuie sur START pendant que la liste est affichée, l'édition d'un nouveau mémo commence.
Si l'on appuie sur START un certain temps, le mémo sélectionné partira à la corbeille.

Lorqu'on édite un fichier BMP quelconque, la sauvegarde se fait au format 24bit sans compression sans tenir compte du format d'origine.

Il est possible d'ouvrir un bmp quelconque si on le place dans le dossier /moonmemo mais comme ce n'est pas prévu pour faire visionneuse, gardez à l'espris que c'est un bloc-notes.
Les fichiers BMP utilisable en tant que mémo, doivent être au formats : Windows, 1/4/8/24bit, 256x192pixels max et sans compression. Les BMP au format OS/2 ne sont pas reconnus.
