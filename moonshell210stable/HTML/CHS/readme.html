・安装 / 卸载 /更新的方法

强烈推荐使用SDFormatter软件格式化SD卡。
安全起见、请给储存卡留下一定的剩余空间（推荐64MByte以上）。

安装：请运行setup.exe。手动拷贝也能安装

卸载：将moonshl2文件夹和moonshl2.nds文件从储存卡里删除。

更新：请运行setup.exe。会将旧版moonshl2备份后在安装。

・无法启动的状况

请确保使用SDFormatter格式化后在安装。
其他的自制应用软件也不能启动的时候，请检查烧录卡内核。
如果只是MoonShell2无法启动、请参考常见问题

・支持的格式

音乐支持： MP1, MP2, MP3, OGG, WAV, M4A, WMA, TTA, MID, RCP, R36, FLAC
MOD支持： MOD, IT, FAR, MTM, S3M, XM, 669, MED, STM, AMF, GDM, ULT, UNI, ASY, IMF, OKT, STX
ChipTune支持： SPC, NSF, GBS, HES, AY, SAP, KSS
播放列表支持：M3U, WPL
图片支持：JPG, BMP, GIF, PNG, PSD
视频支持：DPG
支持文本显示。

・关于格式的限制问题

NDS: 无法运行商业NDS ROM。
WMA: 无法播放无损格式和语音格式。
M4A, WMA: 无法播放经过加密的文件。
MID: 无法播放Standard MIDI file format.2 格式。
SPC: SPC-700模拟的音质相当差。
ChipTune相关: 不兼容FM音源芯片。
播放列表:不支持ANSI, S-JIS, UTF-8, CP437, CP850, CP1252编码格式以外的文件。
播放列表: 无法处理使用"KbMedia Player for Windows Version 2.41.2005.0711"以后版本输出的M3U文件以外的包含多个曲目的ChipTune声音文件的播放列表。
文本: 不支持ANSI, S-JIS, UTF-8, UTF-16BE, UTF-16LE 以外编码。

文件列表菜单中显示文件信息、只支持MP3, Jpeg, BMP, PSD, DPG文件。

・关于大尺寸文件的限制问题

图片: 无法打开超过3648x2736pixels（约1000万像素）的图片。
文本: 没有大小限制、但是超过131072行以后的将不显示。
MOD関係: 无法播放超过500Kbyte的文件。
MID, RCP, R36: 无法播放超过500KByte的文件。

文件夹名和文件名合计超过255字符的（例"/test/mp3files/samplemusic.mp3"合计30个字符）程序不会处理。

・关于高质量文件的限制问题

播放128kbps以上比特率的WMA文件、使用MIDI软音源时和播放SPC时、屏幕显示将简略化。
打开progressive Jpeg需要花费比较长的时间。此过程无法按B中断。（300万像素打开需30秒）
同时进行MIDI播放和图片显示时、可能会发生争夺内存导致声音播放不正常的情况
