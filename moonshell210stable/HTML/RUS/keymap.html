Здесь приведён список действий, выполняемых средствами ввода (кнопки, тач-скрин) в различных режимах.

Функции кнопок в "проводнике" (браузере/списке файлов)
Функции кнопок в "читалке" (Режиме отображения текста)
Функции кнопок в режиме просмотра изображений
Функции кнопок в видео плеере
Функции кнопок в "блокноте"
Список действий кнопок.
Вверх / Вниз / Влево / Вправо (крестовина) Переместить курсор.
Кнопка A Открыть файл.
Кнопка B Остановить музыку. (Если музыка уже остановлена - выход из папки) Выйти из меню
Кнопка X / Y Изменить громкость.
Кнопка START Открыть настройки системы.
Долгое нажатие кнопки START. Удалить текущий файл.
Кнопка SELECT Избранное. (Меню приложений)

Браузер (список файлов). Зажав кнопку L или R...
Клик по экрану Открывается "блокнот" в режиме создания нового файла.

Браузер (список файлов). Зажав кнопку L...
Влево Открыть панель управления MP3.
Вправо Следующий трек.
Вверх Пауза.
Вниз Смена режима (Шаффл, повтор, Повтор всех).
Кнопка A / B Предыдущий или следующий трек.
Кнопка X / Y Сменить уровень подсветки.

Браузер (список файлов). Зажав кнопку R...
Влево / Вправо Перемотка вперёд или назад на 1%.
Вверх / Вниз Перемотка вперёд или назад на 5%.
Кнопки R + A Воспроизвести все треки текущей папки.
Кнопки L + R + A Воспроизвести все треки с флешки.

Браузер (список файлов). Комбинации кнопок L / R.
Одновременное нажатие L + R Пауза.
Краткое + долгое нажатие L / R Перемотка вперёд или назад на 1%.
Двойной клик L / R Предыдущий или следующий трек.
Тройной клик L Пауза.
Тройной клик R Перейти в спящий режим.

Браузер (список файлов). Внешняя кнопка (подключается к разъёму).
Длинное нажатие Перемотка вперёд на 1%.
Краткое + долгое нажатие Перемотка назад на 1%.
Краткое нажатие (клик) Следующий трек.
Двойной клик Предыдущий трек.
Тройной клик Перейти в спящий режим.

Функции кнопок в режиме чтения текста.
Вверх / Вниз Переключение по строкам.
Влево / Вправо Листать страницу.
Кнопка A Листать страницу.
Кнопка B Вернуться в браузер (список файлов).
Кнопка X / Y Открыть меню сохранения/загрузки закладок.
Кнопка START / SELECT Открыть настройки отображения текстов.

Читалка (режим отображения текстов). Зажав кнопку L или R...
Кнопка X / Y Смена уровня подсветки.

Читалка (режим отображения текстов). Комбинации кнопок L / R.
Одновременное нажатие L + R Пауза.
Краткое + долгое нажатие L / R Перемотка вперёд или назад на 1%.
Двойной клик L / R Предыдущий или следующий трек.
Тройной клик L / R Пауза.

Функции кнопок в режиме просмотра изображений.
Вверх / Вниз / Влево / Вправо Перемещение по изображению (Скролл экрана).
Кнопка A / X / Y Изменить зум. (Y=50%, X=100%, A=200%)
Кнопка B Венуться в браузер (список файлов).
Кнопка START Установить изображение в качестве обоев (фонового рисунка).
Кнопка SELECT Открыть настройки изображений.
Зажатие L / R Отображение всего изображения (Отпустите, чтобы вернуться).
