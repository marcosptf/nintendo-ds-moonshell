Come catturare istantanee dello schermo (screenshot).

Per catturare istantanee dello schermo tieni premuto il tasto SELECT per almeno 3 secondi in un momento qualsiasi.
Se la cattura è avvenuta con successo, il NintendoDS si spegnerà automaticamente.
Puoi aprire con Windows il file BMP creato.
Se lo spazio disponibile sulla card non è sufficiente, non verrà salvato alcun file.

Per chi usa un DSi: fare riferimento all'opzione "Attiva la cattura di istantanee con il DSi" nelle impostazioni del Moonshell.
