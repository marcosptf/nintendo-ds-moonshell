2010/01/01 Version 2.09 Stable

시스템부분

  • CycloDS등의 닥터에서 비정상인 섹터를 읽어들일 때 문쉘이 정지하던 호환성 문제를 수정하였습니다.
  • 디스크의 읽기/쓰기 캐쉬에 인터럽트(IRQ)거는 것을 금지함으로 발생하던 소리중단이나 노이즈 현상을 수정하였습니다.
  • 보다 안전하게 인터럽트(IRQ)를 걸거나 해제하게 수정하였습니다.
  • 디스크 엑세싱의 병목현상을 제거하였습니다.
  • Norton Internet Security 프로그램에 보안경고가 되는 문제를 수정하였습니다.

파일목록 부분

  • 확장자가 없거나 0바이트 파일은 표시하지 않도록 수정하였습니다.
  • SELECT 버튼의 선택메뉴에 오프 타이머를 추가하였습니다.
  • moonshl2.ini의 [FileList]섹션에 PowerOffTimerWhileNoInput를 추가하였습니다. 파일목록에서 입력이 없을 경우 꺼질때까지의 시간을 설정할 수 있습니다.

음악재생도구 부분

  • ROMEO2의 음성출력이 좌우가 바뀌어 출력되던 버그를 수정하였습니다.
  • 음악재생목록 파일에서 A버튼을 길게 눌러도 음악파일과 같이 플레이어가 작동되도록 하였습니다.
  • M4A 파일의 모노로된 음성을 재생할 수 없지만 조작 불능이 되지 않게 하였습니다.
  • TTA 파일의[8bits/1ch]과 [8bits/2chs]로 [16bits/1chs]을 재생할 수 없던 버그를 수정하였습니다.
  • moonshl2.ini의 [FileList]섹션의 BButtonToFolderUp가 1 일때 R버튼을 세번 클릭하여 음악을 멈추도록 하였습니다.

문서편집기부분

  • 한줄로만 된 파일을 읽지 못하던 버그를 수정하였습니다.
  • NULL문자를 포함한 파일을 읽지 못하던 버그를 수정하였습니다.

 기타

  • 1.6버전때부터 사용되던 DPGEnc를 DPGTools_AltVer 폴더로 옮겼습니다.
  • 초기에 나온 닌텐도로 문서를 열었을 때 클리어타입 폰트가 제대로 동작하지 않던 버그를 수정하였습니다.

2009/12/02 Version 2.08 Stable

시스템부분

  • 문쉘 시작시에 사용할 수 있는 암호 기능이 추가되었습니다. '/misctools/SetPassword/SetPassword.exe'를 참조해 주세요. 이기능은 사용자의 편리성을 위한 기능일 뿐입니다.
  • MyMeloDS 등을 위해 4032에서 4077KB 크기의 NDS 파일 실행이 가능하게 되었습니다.
  • '/moonshl2/moonshl2.ini' 각 항목에 간단한 영어설명을 곁들였습니다.(한글 설명을 붙이도록 하겠습니다.)
  • DS 브라우저의 확장 메모리를 인식하지 못하던 버그를 수정하였습니다.

파일목록 부분

  • moonshl2.ini 의  [DiskAdapter] 섹션에 파일의 속성에 따라 파일의 표시유무를 설정하는 'HideAttribute_Archive, HideAttribute_Hidden, HideAttribute_System, HideAttribute_ReadOnly' 를 추가하였습니다.

음악재생도구 부분

  • DPG 동영상 플레이어와 거의 비슷한 수준의 음악 플레이어를 추가하였습니다. MP3 파일목록에서 A버튼을 길게 누르시거나 음악 재생중 SELECT버튼을 눌러 선택하실 수 있습니다.
  • 'WMA/32kbps/22kHz' 형식의 음악이 제대로 재생되지 않던 버그를 수정하였습니다.
  • moonshl2.ini 의[PlayList] 섹션에 'IgnorePlayMode_AlwaysOne_FromUserRequest'를 추가하였습니다. 이 셋팅을 1로 하면 음악 재생중 이전곡/다음곡을 선택할 때 재생목록을 무시하고 항상 이전곡/다음곡으로 이동하게 됩니다.
  • 재생모드가 한 곡 반복재생일 때 NSF/GBS/HES/AY/SAP/KSS/SPC등의 총연주 시간을 제대로 인식하지 못했을 시, 자동으로 무한반복 재생을 하도록 하였습니다. 항상 무한반복 재생을 하려면 moonshl2.ini의 [SPCPlugin]섹션의 DefaultLengthSec과 [GMEPlugin]섹션의 DefaultLengthSec을 0으로 변경해 주시기 바랍니다.

문서편집기부분

  • 문서편집기의 수기입력 모델을 조정하였습니다.

  • 상업용 NDS파일을 실행했을 때 에러 메시지는 출력하지만 멈추지 않도록 수정하였습니다.
  • "GBALdr"을 상업용 NDS파일로 잘못 인식해서 실행하지 못하던 버그를 수정하였습니다.

  • moonshl2.ini의[M4APlugin]섹션에 M4A 앨범아트의 배경색을 지정하는 ArtWorkBGColor 옵션을 추가하였습니다. (RRGGBB의 순서로 색을 설정하시면 됩니다.)
  • moonshl2.ini의[M4APlugin]섹션의 ShowCoverImage를[FileList]섹션으로 이동하고 jacket.bmp/mp3 앨범아트/M4A 앨범아트를 ON/OFF 할 수 있도록 수정하였습니다. (이 부분은 설정의 편의를 위해 현재 문라이트씨와 환경설정이나 화면보호기 설정에서 선택할 수 있도록 수정을 하는 방향으로 협의중입니다만, 이 경우 모든 번역자분들에게 양해를 구해야 되는 관계로 시간이 걸릴 듯 합니다.)
  • moonshl2 폴더내에서 START 버튼을 사용한 파일 삭제가 되던 버그를 수정하였습니다. (이 버그에 대해 이메일 주셨던 분께 감사드립니다. 문라이트씨도 나도 전혀 몰랐었음.)
