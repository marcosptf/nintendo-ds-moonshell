Проблемные функции.

Кодировщик видео нестабилен.
Отображение текстов песен [Lyrics (lrc) display] (функция не будет поддерживаться).
Воспроизведение видео форматов, отличных от DPG.

Приложение для Windows не запускается.

Приложение для Windows по MoonShell2 предназначено только для Windows 2000 / XP / Vista.

Муншелл не запускается.

Для некоторых старых адаптеров (флешек).
Пожалуйста, скопируйте файлы 'Clust64k.txt' и 'Ignore16.txt' из папки '/misctools/Additional' в папку '/moonshl2'.
Пожалуйста, воспользуйтесь программой '/misctools/AutoDLDIPatch/AutoDLDIPatch.exe' для создания Moonshell2.nds для адаптеров, не поддерживающих автопатч DLDI.

Файл не открывается.

Файл точно того формата, который понимает MoonShell2?
Расширение файла или его сождержимое точно не отличается от поддерживаемых форматов?

Скин невозможно применить.

Пожалуйста, убедитесь, что версия скин-файла соответствует версии Муншелла (т.е., что скин сделан программой, предназначенной именно для этой версии.
Лог [Illigal ID.] записывается в '/moonshl2/logbuf.txt', когда происходит попытка применить скин более старой версии, более старый/несоответствующий скин игнорируется.

Видео не воспроизводится.

Такие файлы, как avi/mpg/wmv воспроизвести невозможно. Пожалуйста, переконвертируйте подобные файлы в формат DPG.

Не запускается конвертер видео (dpgenc.exe).

Конвертер видео предназначен только для Windows 2000 / XP / Vista.
В других Операционных Системах видео можно конвертировать сторонними приложениями, такими как BatchDPG и Super©.
Я не отвечаю на вопросы по сторонним приложениям.

Видео не конвертируется (dpgenc.exe).

Наиболее вероятно конвертируемый формат видео файлов - MPG.
Файлы других форматов могут конвертироваться некорректно (а то и вовсе могут быть неконвертируемыми). //прим. переводчика

Конвертируются ли файлы, созданные Windows media player?
WMV3 и 120fps не конвертируются.

Видео воспроизводится медленно.

Когда видео подтормаживает - процесс дешифровки происходит несвоевременно.
Когда полоса слева-вверху покраснела - это серьёзно.
Пожалуйста, уменьшите количество кадров в секунду (fps) или битрэйт в настройках конвертации.

Старое видео не воспроизводится.

В случае, если содержится дорожка в формате OGG, файл воспроизвести невозможно.
В случае, если содержится дорожка в формате RGB15/18/21, файл воспроизвести невозможно.
Это очень старые форматы.
