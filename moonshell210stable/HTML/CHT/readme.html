・安裝 / 移除 /更新的方法

強烈推薦使用SDFormatter軟體格式化SD記憶卡。
安全起見、請給記憶卡留下一定的剩餘空間（推薦64MByte以上）。

安裝：請運行setup.exe。手動複製也能安裝

移除：將moonshl2資料夾和moonshl2.nds檔案從記憶卡裡刪除。

更新：請執行setup.exe。會將舊版moonshl2備份後在安裝。

・無法啟動的狀況

請確保使用SDFormatter格式化後再安裝。
其他的自製應用軟體也不能啟動的時候，請檢查燒錄卡內核。
如果只是MoonShell2無法啟動、請參考常見問題

・支援的格式

音樂支援： MP1, MP2, MP3, OGG, WAV, M4A, WMA, TTA, MID, RCP, R36, FLAC
MOD支援： MOD, IT, FAR, MTM, S3M, XM, 669, MED, STM, AMF, GDM, ULT, UNI, ASY, IMF, OKT, STX
ChipTune支援： SPC, NSF, GBS, HES, AY, SAP, KSS
播放列表支援：M3U, WPL
圖片支援：JPG, BMP, GIF, PNG, PSD
螢幕支援：DPG
支援文字文件。

・關於格式的限制問題

NDS: 無法運行商業NDS ROM。
WMA: 無法播放無損格式和語音格式。
M4A, WMA: 無法播放經過加密的檔案。
MID: 無法播放Standard MIDI file format.2 格式。
SPC: SPC-700模擬的音質相當差。
ChipTune相關: 不相容FM音源芯片。
播放列表:不支持ANSI, S-JIS, UTF-8, CP437, CP850, CP1252編碼格式以外的檔案。
播放列表: 無法處理使用"KbMedia Player for Windows Version 2.41.2005.0711"以後版本輸出的M3U檔案以外的、包含多個曲目的ChipTune聲音檔案的播放列表。
文本: 不支持ANSI, S-JIS, UTF-8, UTF-16BE, UTF-16LE 以外編碼。

檔案列表選單中顯示檔案訊息、只支持MP3, Jpeg, BMP, PSD, DPG檔案。

・關於大尺寸檔案的限制問題

圖片: 無法打開超過3648x2736pixels（約1000萬畫素）的圖片。
文本: 沒有大小限制、但是超過131072行以後的將不顯示。
MOD關係: 無法播放超過500Kbyte的檔案。
MID, RCP, R36: 無法播放超過500KByte的檔案。

資料夾名稱和檔案名合計超過255字符的（例"/test/mp3files/samplemusic.mp3"合計30個字元）程式不會處理。

・關於高質量檔案的限制問題

播放128kbps以上位元速度的WMA檔案、使用MIDI軟體音源時和播放SPC時、螢幕顯示將簡單化。
打開progressive Jpeg需要花費比較長的時間。此過程無法按B中斷。（300萬畫素打開需30秒）
同時進行MIDI播放和圖片顯示時、可能會發生爭奪記憶體導致聲音播放不正常的情況
