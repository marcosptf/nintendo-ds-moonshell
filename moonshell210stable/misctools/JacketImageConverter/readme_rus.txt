
Данная утилита создаёт AlbumArt изображение для файлов M4A и MP3.

Поместите файл jacket.bmp в ту же папку, где находится аудиофайл. Данное изображение будет отображаться во время воспроизведения музыки.

Файл Jacket.bmp - ограниченный определённым разрешением BMP файл.
Его можно также создать с помощю любой другой программы-редактора (GIMP или InfanView, например).

jacket.bmp файл должен быть разрешением 256x192pixels, 8bit или 24bit цветов, без компрессии (сжатия), Windows bitmap format.
