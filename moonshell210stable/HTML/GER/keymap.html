Tastenkürzel - Dateiansicht
Oben / Unten / Links / Rechts Cursor bewegen
A Knopf Datei öffnen
B Knopf Musik stoppen
X / Y Knopf Lautstärke ändern
START Knopf Systemmenü öffnen
START Knopf halten Datei löschen
SELECT Knopf öffnet Launcher-Menü

Dateiansicht - L oder R halten +...
Touch screen Erstelle eine neue Notiz.

Dateiansicht - L halten +...
Links Zeige MP3-Toolbar
Rechts Nächster Titel
Oben Pause an/aus
Unten Wechsle Abspielmodus
A / B Knopf Nächst./Vorh. Titel
X / Y Knopf Justiere Helligkeit

Dateiansicht - R halten +...
Links / Rechts Spule 1% Rück-/Vorwärts
Oben / Unten Spule 5% Rück-/Vorwärts
R + A Spiele alle Lieder des Ordners
L + R + A Spiele alle Lieder der Karte

Dateiansicht - L / R Kombinationen
L + R gleichzeitig Pause an/aus
L / R einfach + halten 1% rückw./vorw. spulen
L / R doppelt drücken Vorheriger/Nächster Titel
L dreifach Pause an/aus
R dreifach Standby-Modus

Dateiansicht - Sonstige Funktionen
halten 1% vorspulen
drücken + halten 1% rückspulen
drücken Nächster Titel
doppelt Vorheriger Titel
dreifach Standby-Modus
