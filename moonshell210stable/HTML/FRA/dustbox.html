Utilisation de la Corbeille

Lorsque vous créez un dossier nommé /dustbox à la racine du disque, il devient possible de supprimer des fichiers (déplacer à la corbeille).

Pour supprimer un fichier, placez le curseur sur celui-ci puis pressez START un certain temps. Un message de confirmation apparaît. Dès lors, pressez les boutons L+R+A pour supprimer ce fichier. Autrement, pressez le bouton B pour annuler.
Pour éviter tout problème, les fichiers ne sont pas vraiment supprimés mais déplacés vers le dossier /dustbox. Le contenu de la corbeille ne peut être effacé qu'à l'aide de votre ordinateur.
Pour vider la corbeille de Moonshell, n'utilisez pas la commande "Vider la cobeille" de votre ordinateur. Sélectionnez les fichiers à supprimez puis supprimez-les.
Lorsque l'on jette un fichier qui a le même nom qu'un autre se trouvant dans le dossier /dustbox, l'ancien sera écrasé.

Si l'on essaie de supprimer un dossier, un fichier de la corbeille ou fichier du dossier /moonshl2, un message d'érreur apparaît puis l'opération est annulée.

Il est possible de masquer le dossier de la corbeille (afin que MoonShell2 n'affiche pas ce dossier) en changeant un paramètre du fichier /moonshl2/hidepath.ini

Pour cela, remplacez :
;/dustbox
par
/dustbox

Il est possible de lire les fichiers qui se trouvent dans la corbeille.

Si vous supprimer souvent des fichiers et que vous trouvez gênant de devoir presser L+R+A à chaque fois, veuillez vous référer à /misctools/Additional/EasyDele.txt.
