Funcionalidades abandonadas

Letras para canciones MP3 (lrc).
Analizador de espectro, archivo de parámetros, lista de reproducción y editor de ecualización.
Lectura de texto vertical.
El pequeño ruido justo después de reproducir un archivo de sonido o de iniciar el protector de pantalla.
Lectura de archivos de vídeo distintos de DPG.

Las herramientas previstas para Windows no pueden iniciarse.

Las herramientas proporcionadas por MoonShell2 son para Windows2000/XP/Vista.
No han sido probadas en Windows95/98/Me. Creo que no funcionarán.

Moonshell2 no se inicia.

Si la tarjeta tiene clusters de 64KB, por favor refiérase a Clust64k.txt e Ignore16.txt en la carpeta /misctools/Additional/.
Utilice /misctools/AutoDLDIPatch/AutoDLDIPatch.exe. Puede que su controlador no esté actualizado.
Las flashcart EZ4, R4SDHC y ahora M3Sakura están excluidas por software.

Los archivos no se abren.

Por favor, compruebe que el formato de archivo es reconocido por MoonShell2.
Es posible que la extensión sea la misma, pero el formato sea diferente. Moonshell2 no puede abrir archivos de gran tamaño.
Por favor, lea la lista de formatos copmpatibles

No puede aplicarse un skin.

Por favor, compruebe que el archivo corresponde a la versión actual de Moonshell.
Si intenta cargar un skin de una versión anterior, se escribirá el error [Illegal ID.] en /moonshl2/logbuf.txt.

No pueden verse los vídeos.

Los archivos avi/mpg/wmv no pueden ejecutarse. Sólo el formato DPG es reproducible.

El codificador de vídeo (dpgenc.exe) no funciona.

El codificador de vídeo está previsto para Windows2000/XP/Vista.
No funcionará sobre Windows95/98/Me.
Es posible que los vídeos codificados por otro programa (BatchDPG, etc..) no puedan reproducirse.
No puedo responder a preguntas acerca de otros codificadores.

No puede recodificarse el vídeo.

MPG es el formato más fácil de convertir.
Si tiene problemas para convertir de AVI/WMV, trate de convertir primero a MPG con otro conversor.
Personalmente, el único formato que he intentado convertir directamente es el de Youtube.

Los formatos especiales (MP4, DivX, OGM, etc ...) reproducibles con Windows Media Player pueden ser convertidos.
Reproduzca los vídeos con Windows Media Player antes de la conversión.
Sin embargo, los WMV3 no son compatibles.
Los vídeos con más de 120fps no pueden ser convertidos.

Hay ralentizaciones en el vídeo.

Después de mover el cursor de la barra de tiempo durante la reproducción, pueden aparecer problemas de reposicionamiento.
Además, si la pequeña barra horizontal, que aparece de vez en cuando a la izquierda en la barra de tiempo, se vuelve de color rojo, es que el vídeo es demasiado grande.
En ese caso, vuelva a codificar el vídeo con menos fps.

El sonido pierde sincronía con la imagen después de mover el cursor en la barra de tiempo.

No se puede hacer mucho. Trate de mover el cursor varias veces, y con suerte se resincronizará.

Los vídeos antiguos no funcionan.

Los vídeos DPG con archivos de sonido en formato OGG streaming o la imagen RGB15/18/21 no son compatibles con Moonshell2.

Se cuelga en el mensaje "no se puede encontrar la carpeta moonshl." durante el inicio.

Es un mensaje de error del antiguo Moonshell. No es necesario desinstalar los antiguos Moonshell porque pueden coexistir. Asegúrese de que ejecuta el archivo "moonshl2.nds".

Moonshell2 no se inicia en R4.

Puede haber varias razones, como que el sistema de su tarjeta no esté actualizado. Las versiones antiguas pueden no parchear automáticamente el controlador DLDI.
Por favor, compruebe que su tarjeta está actualizada.

El homebrew no se inicia en R4

Los hombrew iniciados desde MoonShell2 en un R4 (con la aplicación MS2 original) no funcionan; el uso de la pantalla táctil puede ser un problema.
Honestamente, no puedo identificar el problema que les impide funcionar. ¿Cómo se puede solucionar?

Moonshell2 no se inicia en AceKard2

El firmware AceKard2_v4.16_menu y AKAIO_ak2loader_2009.04.03 producen un error grave al cargar MoonShell2.
Use un firmware 4.15 o anterior, o un AKAIO_ak2loader más reciente.

Moonshell2 no se inicia en SuperCard con soporte GBA

La función Auto DLDI puede causar algún problema con el parche
Copie y elimine de la raiz el archivo dldi.scp (si existe) antes usar el programa con el parche DLDI aplicado con Windows.
