Некоторые полезные сведения, которые могут помочь при создании skn-файлов (так называемых, "тем" или "скинов"):

______
Правило именования файла default.skn.

При использовании CP000(ENG). MoonShell2 ищет файл def_000.skn.
При использовании CP866(RUS). MoonShell2 ищет файл def_866.skn.
При использовании CP932(JPN). MoonShell2 ищет файл def_932.skn.

Используемая Муншеллом кодировка соответствует названию файла.
CP866 - кодировка для русского языка.

Если специализированный для языка файл не найден, используется default.skn.

______
Все PNG файлы позволяют использовать прозрачность, регулируемую от 0% до 100%.
Настройка уровня прозрачности особенно полезна для файлов FL_DOUBLE_ITEMBG_CLEAR.PNG и FL_SINGLE_ITEMBG_CLEAR.PNG, чтобы не скрыть полностью выбор файла в проводнике.

Уровень прозрачности элемента влияет на скорость отображения.
0% (полностью видимый) - отображается быстрее всего.
Почти также быстро отображается элемент 100% уровня (полностью прозрачный).
Следом идут элементы со значением прозрачности в 6,25% / 12,5% / 25% / 50%. 
Они отображаются практически с одинаковой скоростью.
Остальные уровни прозрачности отображаются медленнее всего.
Если изображение имеет несколько уровней прозрачности,
меняющихся по горизонтали, отображение будет наиболее медленным.
Смена уровня прозрачности изображения по вертикали
на скорости отображения не сказывается.

Использование полностью прозрачных или полностью непрозрачных изображений
с минимальным возможным количеством прозрачных цветов, позволяет создать быстрый скин.
Однако, файлы, кроме FL_DOUBLE_ITEMBG_CLEAR.PNG и FL_SINGLE_ITEMBG_CLEAR.PNG, 
не должны значительно влиять на скорость отображения.

Файлы BMP могут отображаться немного медленнее, но время для отображения
не зависит от того, насколько сложным является изображение.
