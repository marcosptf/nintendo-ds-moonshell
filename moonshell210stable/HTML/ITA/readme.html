Grazie per avere scaricato MoonShell 2.
Per eseguire l'applicazione copia il file 'moonshl2.nds' e la cartella '/moonshl2' nella root della tua MicroSD. Una volta fatto, puoi avviare il file 'moonshl2.nds'.

Formati di file supportati:

Formati audio: MP1, MP2, MP3, OGG, WAV, M4A, AAC, WMA, TTA, MID, RCP, R36, FLAC
Formati MOD: MOD, IT, MTM, S3M, XM, 669, MED, STM, AMF, GDM, ULT, UNI, ASY, IMF, OKT, STX
Formati Chiptune: SPC, NSF, GBS, HES, AY, SAP, KSS
Formati Playlist: M3U, WPL
Formati immagine: JPG, BMP, GIF, PNG, PSD
Formati video: DPG
Supporta i file di testo.

Limitazioni per alcuni formati:

NDS: Le ROM NDS commerciali non sono supportate.
WMA: Non è supportato il modo "lossless" (senza perdita) e il "modo voce".
M4A, AAC, WMA: Non sono supportati i file crittografati.
MID: Il formato 'Standard MIDI 2' non è supportato.
SPC: La qualità del tono è molto scadente.
Chiptune: Viene avviata solo la prima traccia. Il suono FM non viene emulato.
Playlist: Le codifiche supportate sono ANSI, S-JIS, UTF-8, CP437, CP850, CP1252.
Text file: Le codifiche supportate sono ANSI, S-JIS, UTF-8, UTF-16BE, UTF-16LE.

Nella lista file sono mostrate solo le informazioni dei formati MP3, Jpeg, BMP, PSD, DPG.

Limitazioni per file di grandi dimensioni:

Formati immagine: Non possono essere visualizzate immagini con risoluzione maggiore di 3648x2736 pixel.
File di testo: Non possono essere visualizzate le righe oltre la n° 131072.
MOD: Un file più grande di 500KByte potrebbe non essere riprodotto.
MID, RCP, R36: Un file più grande di 500KByte potrebbe non essere riprodotto.
