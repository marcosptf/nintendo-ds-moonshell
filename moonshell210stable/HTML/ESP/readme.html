Instalación / Desinstalación

En Windows, se recomienda formatear la tarjeta de memoria con SDFormatter.
Asegúrese de dejar por lo menos 32MB de espacio libre en la tarjeta para que todo funcione bien.

En general, para instalar Moonshell 2, simplemente copie el archivo /moonshl2 y el archivo moonshl2.nds en la raíz de la tarjeta.
Para desinstalar, borre la carpeta /moonshl2 y el archivo moonshl2.nds de la tarjeta.

Por favor, consulte Método de instalación con imágenes para más detalles.

Qué hacer si no se inicia

Verifique que la carpeta /moonshl2 se encuentra en la raíz de la tarjeta.
Si no se ejecutan los homebrews compruebe el formato de la tarjeta o el firmware, por ejemplo.
Si sólo es Moonshell 2 el que no se inicia, por favor, consulte Preguntas frecuentes

Lista de formatos compatibles

Audio:
Formatos tipo Compresión : MP1, MP2, MP3, OGG, WAV, M4A, AAC, WMA, TTA, MID, RCP, R36 y FLAC.
Formatos tipo Módulo : MOD, IT, MTM, S3M, XM, 669, MED, STM, AMF, GDM, ULT, UNI, ASY, IMF, OKT y STX.
Formatos tipo ChipTune : SPC, NSF, GBS, HES, AY, SAP y KSS.

Listas de reproducción : M3U y WPL.

Imágenes : JPG, BMP, GIF, PNG y PSD.

Vídeos : DPG y DPG2.

Documentos : TXT.

Limitaciones con determinados formatos

NDS : Los archivos NDS de ROM comerciales no se pueden ejecutar.
WMA : El modo sin pérdida y el modo de voz no se pueden reproducir.
M4A, AAC, WMA : Los archivos con DRM no se pueden reproducir.
MID : Los archivos MIDI en formato 2 no se pueden reproducir.
SPC : La emulación del RCP-700 proporciona una calidad de sonido muy pobre.
Los tipos ChipTune : Sólo se reproduce la primera pista de los archivos que contienen varias. Además, el sonido FM no se puede generar.
Lista de reproducción : Sólo las codificaciones ANSI, S-JIS, UTF-8, CP437, CP850 y CP1252 son reconocidas.
Textos: Sólo las codificaciones ANSI, S-JIS, UTF-8, UTF-16BE y UTF-16LE son reconocidas.

Sólo la información de los formatos MP3, JPEG, BMP, PSD y DPG son mostrados en el navegador de archivos.

Restricciones en el tamaño de los archivos

Imágenes : Puede mostrar las imágenes (excepto GIF) hasta 3648x2736pixels (10 mega píxeles).
Textos : No hay límite de tamaño para archivos de texto (confirmado hasta 8MB).
Los tipos Módulo : No se puede reproducir la mayoría de los archivos de más de 500kb.
MID, RCP, R36 : No se puede reproducir la mayoría de los archivos de más de 500kb.

Límites para los archivos grandes

Se ha simplificado la pantalla para la reproducción de archivos WMA de más de 128 kbps, para el uso de instrumentos por software MIDI y para la reproducción de SPC, ya que consumen una gran cantidad de recursos.
Los JPEG tardan mucho tiempo en aparecer (30 segundos por cada 3 mega píxeles).
Pueden ocurrir muchos parones al reproducir un archivo MIDI y mostrar una imagen al mismo tiempo.
