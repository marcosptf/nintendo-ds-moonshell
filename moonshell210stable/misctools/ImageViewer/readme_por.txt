----------

- MakeNDSROM.exe
Cria o arquivo NDSROM para cada adaptador. 

- imgview.ini
Este é um arquivo de configurações.
Quando o 'Image viewer' é usado com a configuração padrão, este arquivo é desnecessário.

- img2ipk.exe
Duas ou mais imagens são inseridas no arquivo. As miniaturas são criadas.

----------

Por favor execute o 'MakeNDSROM.exe'. Então, copie o arquivo NDSROM de seu adaptador para o cartão.

O trabalho de conversão é necessário para as imagens.

Por favor mude a opção 'customjpeg mode' para 'ZLIB mode' para uma melhor qualidade nas imagens. O tamanho do arquivo ficará maior.

----------

'Image viewer' funciona melhor com imagens grandes quando existe memória cache.
A utilização em conjunto da memória do DS Browser ou do EZ3in1 é recomendado. juntos são recomentados.
'O cartucho de expansão de memória do DS Browser' ou o 'EZ 3in1 Expansion Pack' são recomendados.
Há uma ferramenta para teste de memória na quarta página das opções do slide show.

----------

'Cima/Baixo/Esquerda/Direita/L' e 'A/B/X/Y/R' possuem a mesma função.
Todo o trabalho pode ser feito através dos ícones e da Touch Screen.

Modo de seleção de arquivo IPK.

Ícone superior esquerdo : Muda o nível de brilho.
Cima/Baixo : Move o cursor.
L: Decisão. (modo miniatura)
Start : Soft reset ou, move para as opções do slide show.

Modo miniatura.

Ícone superior esquerdo : Muda o nível de brilho.
Ícone superior direito : Volta para a seleção de arquivo IPK.
Ícone superior centro : Muda entre vertical e horizontal.
Ícone inferior centro : Move para as opções de slide show.
Cima/Baixo/Esquerda/Direita : Move o cursor.
L : Decisão. (modo visualização)
Start : Soft reset ou, move para as opções de slide show.
Select : Volta para a seleção de arquivo IPK.

Modo opções de slide show.

Start : Soft reset ou, início de slide show.

Modo de visualização.

Ícone superior esquerdo : Muda o nível de brilho.
Ícone superior direito : Volta para o modo miniatura.
Ícone inferior esquerdo : Imagem anterior.
Ícone inferior direito : Próxima página.
Ícone inferior centro esquerdo : Muda entre vertical e horizontal.
Ícone inferior centro direito : Muda entre 'Single screen and preview mode' e 'Double screen mode'.
Ícone inferior centro : Altera a taxa de redução.
Cima/Baixo/Esquerda/Direita : Move o cursor.
Start : Soft reset ou, move para as opções de slide show.
L : Menu de comando.
L + Cima : Volta para modo miniatura.
L + Esquerda : Imagem anterior.
L + Direita : Próxima página.
L + Select : Posição inicial do menu .
