
Поддерживается только 1 формат: WAVE 8бит, без компрессии.

snd_click.wav = Звук, воспроизводимый при нажатии на кнопку.
snd_movepage.wav = Звук, воспроизводимый при смене страницы, переходу в папку, и т.д...
snd_notify.wav = Звук, воспроизводимый при показе окна информации.
snd_open.wav = Звук, воспроизводимый при открытии.
snd_longtap.wav = Звук, воспроизводимый при задержании стилуса на определённой точке некоторое время (при "долгом клике стилусом").
snd_poweroff.wav = Звук, воспроизводимый при автоматическом выключении NDS.
