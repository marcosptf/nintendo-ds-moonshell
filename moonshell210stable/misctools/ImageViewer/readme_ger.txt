----------

- MakeNDSROM.exe
Die NDSROM-Datei für jeden Adapter wird erstellt.

- imgview.ini
Dies ist die Konfigurationsdatei.
Wenn 'Image viewer' mit den Standardeinstellungen genutzt wird, wird diese nicht benötigt.

- img2ipk.exe
Zwei oder mehr Bilder werden gepackt. Ein Vorschaubild wird erstellt.

----------

Bitte starten Sie 'MakeNDSROM.exe'. Bitte kopieren Sie das NDSRom Ihres Adapters auf die Karte.

Die Konvertierung wird für die Bilddatei benötigt.

Bitte wechseln Sie von 'customjpeg mode' zu 'ZLIB mode' für eine höhere Bildqualität. Die Dateigröße steigt dadurch.

----------

'Image viewer' unterstützt schnelles scrollen in großen Bildern falls eine Speichererweiterung vorhanden ist.
Die Nutzung des DS Browser Memory oder EZ3in1 wird empfohlen.
Auf der vierten Seite des Reiters "Show options" steht ein Tool zum testen des Speichers zur verfügung.

----------

'Oben/Unten/Links/Rechts/L-Taste' und 'A/B/X/Y/R-Taste' haben die selben Funktionen.
Jede Arbeit kann grundlegend auch durch die Nutzung der Icons und des Touchscreens erledigt werden.

IPK Dateiauswahl

Icon Links oben : Ändert den Bildschirmkontrast
Oben / Unten Knopf : Bewegt den Cursor
L-Taste : Auswählen (zum Thumbnail-Modus)
START-Knopf : Soft Reset, oder wechseln in den Slideshow Setting-Modus

Thumbnail-Modus

Icon Links oben : Ändern den Bildschirmkontrast
Icon Rechts oben : Zurück zur IPK Dateiauswahl
Icon Mitte oben : Wechseln zwischen horizontal und vertikal
Icon Mite unten : Zum Slideshow Setting-Modus
Oben/Unten/Links/Rechts Knopf : Bewegt den Cursor
L-Taste : Auswählen (zum Darstellungs-Modus)
START-Knopf : Soft Reset, oder wechseln in den Slideshow Setting-Modus
SELECT-Knopf : Zurück zur IPK Dateiauswahl

Slideshow Setting-Modus

START-Knopf : Soft Reset, oder Slideshow starten

View mode.

Icon Links oben: Ändern den Bildschirmkontrast
Icon Rechts oben : Zurück zum Thumbnail-Modus
Icon Links unten : Zum vorherigen Bild
Icon Rechts unten : Zur nächsten Seite
Icon Mitte links oben : Wechseln zwischen horizontal und vertikal
Icon Mitte rechts oben : Wechsle zwischen 'Ein-Bild + Vorschau Modus' und 'Zwei-Bild-Modus'
Icon Mitte unten : Die Kompressionsrate wird geändert
Oben/Unten/Links/Rechts Knopf : Bewegt den Cursor
START-Knopf : Soft Reset, oder wechseln in den Slideshow Setting-Modus
L-Taste : Öffnet das erweiterte Kontrollmenü
L-Taste + Oben : Zurück zum Thumbnail-Modus
L-Taste + Links : Zum vorherigen Bild
L-Taste + Rechts : Zur nächsten Seite
L-Taste + SELECT-Knopf : Startpositionsmenü wird geöffnet
